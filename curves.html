Krzywe

Łuk

Aby utworzyć łuk, używamy metody arc(). Łuki są definiowane przez punkt środkowy, promień, kąt początkowy, kąt końcowy i kierunek rysowania (zgodnie lub przeciwnie do ruchu wskazówek zegara). Łuki mogą być stylizowane za pomocą właściwości lineWidth, strokeStyle i lineCap.

Łuk to nic innego jak wycinek obwodu koła. Ten okrąg definiujemy przez x, y i promień. Następnie definiujemy sam łuk za pomocą dwóch punktów na obwodzie okręgu zdefiniowanego przez startAngle i endAngle. Te dwa kąty są określone w radianach i tworzą linie wychodzące ze środka okręgu i przecinające końce łuku, który chcemy utworzyć. Ostatnim argumentem metody arc() jest antiClockwise, który określa kierunek ścieżki łuku między dwoma jej punktami końcowymi. Ten argument ma domyślną wartość false, co powoduje, że łuk jest rysowany zgodnie z ruchem wskazówek zegara.

<canvas id="firstCanvas" width="300px" height="300px"></canvas>
canvas {
    border: 3px solid black;
}
const canvas = document.getElementById("firstCanvas");
const ctx = canvas.getContext("2d");
        
let x = canvas.width / 2;
let y = canvas.height / 2;
let radius = 75;
let startAngle = 1.1 * Math.PI;
let endAngle = 1.9 * Math.PI;
let counterClockwise = false;

ctx.beginPath();
ctx.arc(x, y, radius, startAngle, endAngle, counterClockwise);
ctx.lineWidth = 15;
ctx.strokeStyle = "black";
ctx.stroke();

Parabola

Aby utworzyć parabolę, używamy metody quadraticCurveTo(). Parabole są definiowane przez punkt kontekstowy, punkt kontrolny i punkt końcowy. Można je stylizować za pomocą właściwości lineWidth, strokeStyle i lineCap.

Punkt kontrolny definiuje krzywiznę paraboli, tworząc dwie linie styczne, które są połączone z punktem kontekstowym i punktem końcowym. Punkt kontekstu jest definiowany przez metodę moveTo(). Przesunięcie punktu kontrolnego dalej od punktu kontekstu i punktu końcowego spowoduje utworzenie ostrzejszych krzywych, a przesunięcie punktu kontrolnego bliżej punktu kontekstu i punktu końcowego stworzy szersze krzywe.

<canvas id="secondCanvas" width="300px" height="300px"></canvas>
canvas {
    border: 3px solid black;
}
const canvas = document.getElementById("secondCanvas");
const ctx = canvas.getContext("2d");
        
ctx.beginPath();
ctx.moveTo(75, 150);
ctx.quadraticCurveTo(150, 30, 225, 150);
ctx.lineWidth = 10;
ctx.strokeStyle = "black";
ctx.stroke();

Krzywa Beziera

Aby utworzyć krzywą Beziera, używamy metody bezierCurveTo(). Krzywe Beziera są definiowane za pomocą punktu kontekstowego, dwóch punktów kontrolnych i punktu końcowego. W przeciwieństwie do paraboli, krzywe Beziera są definiowane za pomocą dwóch punktów kontrolnych zamiast jednego, co pozwala nam tworzyć bardziej złożone krzywizny. Krzywe Beziera można stylizować za pomocą właściwości lineWidth, strokeStyle i lineCap.

Krzywa Beziera jest zdefiniowana przez bieżący punkt kontekstu, dwa punkty kontrolne i punkt końcowy. Pierwsza część krzywej jest styczna do linii zdefiniowanej przez punkt kontekstowy i pierwszy punkt kontrolny. Druga część krzywej jest styczna do linii określonej przez drugi punkt kontrolny i punkt końcowy.

<canvas id="thirdCanvas" width="300px" height="300px"></canvas>
canvas {
    border: 3px solid black;
}
const canvas = document.getElementById("thirdCanvas");
const ctx = canvas.getContext("2d");
        
ctx.beginPath();
ctx.moveTo(30, 210);
ctx.bezierCurveTo(60, 60, 270, 60, 240, 210);
ctx.lineWidth = 10;
ctx.strokeStyle = "black";
ctx.stroke();

Ćwiczenie

Stwórz dowolną ścieżkę składającą się z łuku, paraboli i krzywej Beziera.

<canvas id="fifthCanvas" width="300px" height="300px"></canvas>
canvas {
        border: 3px solid black;
    }
const canvas = document.getElementById("fifthCanvas");
const ctx = canvas.getContext("2d");
            
ctx.beginPath();
ctx.moveTo(30, 30);
ctx.lineTo(90, 150);
ctx.quadraticCurveTo(120, 30, 150, 150);
ctx.bezierCurveTo(210, 180, 240, 210, 150, 240);
ctx.lineWidth = 10;
ctx.stroke();
