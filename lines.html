Linie

Linia

Aby narysować linię, korzystamy z metod beginPath(), moveTo(), lineTo() i stroke().

Na początek używamy metody beginPath(), aby utworzyć nową ścieżkę. Następnie metody moveTo() do ustawienia punktu kontekstowego (tj. kursora rysowania). Potem metody lineTo() do narysowania linii prostej od pozycji początkowej do pozycji końcowej. Na koniec, aby linia była widoczna, ustawiamy obrys linii za pomocą metody stroke(). Domyślnym kolorem obrysu jest czarny. 

<canvas id="firstCanvas" width="300px" height="300px"></canvas>
canvas {
    border: 3px solid black;
}
const canvas = document.getElementById("firstCanvas");
const ctx = canvas.getContext("2d");
        
ctx.beginPath();
ctx.moveTo(50, 75);
ctx.lineTo(250, 250);
ctx.stroke();

Szerokość linii

Aby zdefiniować szerokość linii, używamy właściwości lineWidth. Ta właściwość musi być ustawiona przed wywołaniem metody stroke().

<canvas id="secondCanvas" width="300px" height="300px"></canvas>
canvas {
    border: 3px solid black;
}
const canvas = document.getElementById("secondCanvas");
const ctx = canvas.getContext("2d");
        
ctx.beginPath();
ctx.moveTo(100, 150);
ctx.lineTo(250, 50);
ctx.lineWidth = 15;
ctx.stroke();

Kolor linii

Aby ustawić kolor linii, używamy właściwości strokeStyle, którą można ustawić na nazwę koloru, taką jak red, green lub blue, wartość szesnastkową, jak #FF0000 lub #555, lub wartość RGB, na przykład rgb(255, 0, 0).

<canvas id="thirdCanvas" width="300px" height="300px"></canvas>
canvas {
    border: 3px solid black;
}
const canvas = document.getElementById("thirdCanvas");
const ctx = canvas.getContext("2d");
        
ctx.beginPath();
ctx.moveTo(75, 200);
ctx.lineTo(150, 50);
ctx.lineWidth = 10;
ctx.strokeStyle = "#ff0000";
ctx.stroke();

Zakończenie linii

Aby zmienić zakończenie linii, używamy właściwości lineCap. Linie mogą mieć jeden z trzech stylów: butt, round, lub square. Domyślny styl zakończenia to butt. Właściwość lineCap należy ustawić przed wywołaniem stroke().

<canvas id="fourthCanvas" width="300px" height="300px"></canvas>
canvas {
    border: 3px solid black;
}
const canvas = document.getElementById("fourthCanvas");
const ctx = canvas.getContext("2d");
        
ctx.beginPath();
ctx.moveTo(250, 100);
ctx.lineTo(50, 150);
ctx.lineWidth = 10;
ctx.strokeStyle = "green";
ctx.lineCap = "round";
ctx.stroke();
