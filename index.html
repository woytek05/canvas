Jak zacząć?

1. Umieść element <canvas> w dokumencie HTML. Pamiętaj o zamknięciu znacznika. Zawartość w środku wyświetli się, jeśli canvas nie jest wspierany przez przeglądarkę. Jest to tzw. fallback content. 

<canvas id="canvas"></canvas>

2. Złap go za pomocą JavaScriptu. 

const canvas = document.getElementById("canvas");

3. Utwórz kontekst dla tego elementu. 

const canvas = document.getElementById("canvas");
const ctx = canvas.getContext("2d");

Podczas korzystania z canvasa ważne jest, aby zrozumieć różnicę między samym canvasem a jego kontekstem. Element <canvas> to węzeł DOM osadzony na stronie HTML. Kontekst canvasa to obiekt z właściwościami i metodami, których można użyć do renderowania grafiki wewnątrz tego elementu. Kontekst może być 2d lub 3d.
