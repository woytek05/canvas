Kształty

Niestandardowe kształty

Aby utworzyć niestandardowy kształt, możemy utworzyć ścieżkę lub krzywej do skonstruowania każdej podścieżki, która tworzy nasz kształt. 

<canvas  id="myCanvas2" width="300" height="300"></canvas>
const canvas1 = document.getElementById("myCanvas1");
const ctx1 = canvas1.getContext("2d");

ctx1.beginPath();
ctx1.moveTo(170, 80);
ctx1.bezierCurveTo(130, 100, 130, 150, 230, 150);
ctx1.bezierCurveTo(250, 180, 320, 180, 340, 150);
ctx1.bezierCurveTo(420, 150, 420, 120, 390, 100);
ctx1.bezierCurveTo(430, 40, 370, 30, 340, 50);
ctx1.bezierCurveTo(320, 5, 250, 20, 250, 50);
ctx1.bezierCurveTo(200, 5, 150, 20, 170, 80);

ctx1.closePath();
ctx1.lineWidth = 5;
ctx1.stroke();

Prostokąt

Do utworzenia prostokątu, możemy użyć metody rect() zamiast konstruować kształt za pomocą 4 linii łączących. Prostokąt jest pozycjonowany za pomocą parametrów x i y, a jego rozmiar odpowiada parametrom szerokości i wysokości.
Prostokąt jest umieszczony względem lewego górnego rogu.

Przykład:

<canvas  id="myCanvas2" width="300" height="300"></canvas>
var canvas2 = document.getElementById('myCanvas2');
var ctx2 = canvas2.getContext('2d');

ctx2.beginPath();
ctx2.rect(50, 50, 200, 100);
ctx2.fillStyle = 'white';
ctx2.fill();
ctx2.lineWidth = 5;
ctx2.strokeStyle = 'grey';
ctx2.stroke();

Koło

Następny kształt do narysowania to koło.
Możemy utworzyć pełny łuk za pomocą metody arc(), definiując kąt początkowy jako 0 i kąt końcowy jako 2 * PI.

<canvas id="myCanvas" width="300" height="300" ></canvas>
var canvas = document.getElementById('myCanvas');
var ctx3 = canvas.getContext('2d');
var centerX = canvas.width / 2;
var centerY = canvas.height / 2;
var radius = 70;

ctx3.beginPath();
ctx3.arc(centerX, centerY, radius, 0, 2 * Math.PI, false);
ctx3.fillStyle = 'grey';
ctx3.fill();
ctx3.lineWidth = 5;
ctx3.strokeStyle = 'black';
ctx3.stroke();

Półkole

By stworzyć półkole, możemy utworzyć łuk za pomocą metody arc() i zdefiniować kąt końcowy jako startAngle + PI.

Poniżej przykłąd:

<canvas id="myCanvas" width="300" height="300" ></canvas>
var canvas4 = document.getElementById('myCanvas4');
var ctx4 = canvas4.getContext('2d');

ctx4.beginPath();
ctx4.arc(150, 100, 70, 0, Math.PI, false);
ctx4.closePath();
ctx4.lineWidth = 5;
ctx4.fillStyle = 'darkgray';
ctx4.fill();
ctx4.strokeStyle = 'black';
ctx4.stroke();
