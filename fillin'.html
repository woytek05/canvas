Style wypełnienia

Wypełnienie kolorem oraz obramowanie

Do wypełniania kształtu jednolitym kolorem, możemy ustawić właściwość fillStyle na ciąg koloru, taki jak niebieski, wartość szesnastkową, taką jak #0000FF,
lub wartość RGB, taką jak rgb(0,0,255), a następnie możemy użyj metody fill(), aby wypełnić kształt. O ile nie określono inaczej, domyślny styl wypełnienia kształtu jest czarny. 

<canvas  id="myCanvas2" width="300" height="300"></canvas>
const canvas1 = document.getElementById('myCanvas1');
const ctx1 = canvas1.getContext('2d');
const centerX = canvas1.width / 2;
const centerY = canvas1.height / 2;
const radius = 70;

ctx1.beginPath();
ctx1.arc(centerX, centerY, radius, 0, 2 * Math.PI, false);
//dajemy kolor, którym chcemy wypełnić środek
ctx1.fillStyle = 'blue'
ctx1.fill();
ctx1.lineWidth = 5;
//dodajemy kolor obrysu
ctx1.strokeStyle = '#ff0000'
ctx1.stroke();

Do wypełnienia obramowania kształtu używamy komendy strokeStyle() i podajemy wartość, a następnie używamy komendy stroke(). Warto pamiętać aby funkcję powyższą funkcję stosować po funkcji fill(). W przeciwnym razie wypełnienie nałoży się na połowę obrysu.

Gradient liniowy

Aby stworzyć gradient liniowy w canvasie, możemy użyć metody createLinearGradient().
Gradienty liniowe są definiowane przez wyimaginowaną linię, która określa kierunek gradientu.
Po utworzeniu gradientu możemy wstawić kolory za pomocą właściwości addColorStop. 

<canvas  id="myCanvas2" width="300" height="300"></canvas>
const canvas2 = document.getElementById('myCanvas2');
const ctx2 = canvas2.getContext('2d');
ctx2.rect(0, 0, canvas2.width, canvas2.height);

//Tu tworzymy gradient
const grd = ctx2.createLinearGradient(0, 0, canvas2.width, canvas2.height);
//dajemy kolor początkowy
grd.addColorStop(0, '#00ffFF');   
//dajemy kolor końcowy
grd.addColorStop(1, 'darkblue');
ctx2.fillStyle = grd;
ctx2.fill();

Kierunek gradientu liniowego przesuwa się od punktu początkowego do punktu końcowego wyimaginowanej linii zdefiniowanej za pomocą metody createLinearGradient(). Używamy dowolnego koloru który rozpoczyna się w punkcie początkowym gradientu i drugi kolor, który kończy się punktem końcowym. Ograniczenia kolorów są umieszczone wzdłuż wyobrażonej linii gdzieś pomiędzy 0 a 1, gdzie 0 to punkt początkowy, a 1 to punkt końcowy.

Gradient od promienia

Do stworzenia gradientu promieniowego, możemy użyć metody createRadialGradient(). Gradienty promieniowe są definiowane za pomocą dwóch wyimaginowanych okręgów - okręgu początkowego i okręgu końcowego, w których gradient zaczyna się od okręgu początkowego i przesuwa się w kierunku okręgu końcowego.

Przykład:

<canvas  id="myCanvas2" width="300" height="300"></canvas>
var canvas3 = document.getElementById('myCanvas3');
var ctx3 = canvas3.getContext('2d');
ctx3.rect(0, 0, canvas3.width, canvas3.height);
//Odstep od lewej , odstep od gory , nasilenie koloru , promień o który okrąg się obraca, rozszerzenie okręgu o promień, promień gradientu
var grd = ctx3.createRadialGradient(15, 200, 0, 100, 50, 300);
grd.addColorStop(0, '#00ffff');
grd.addColorStop(1, 'darkblue');

ctx3.fillStyle = grd;
ctx3.fill();

Wzór

Tworząc pattern, możemy użyć metody createPattern(), która zwraca obiekt wzorca. Ustawiamy właściwość fillStyle na obiekt wzorca, a następnie wypełnić kształt za pomocą funkcji fill().
Metoda createPattern() wymaga obiektu obrazu i opcji powtórki, którą może być repeat, repeat-x, repeat-y lub no-repeat. O ile nie określono inaczej, opcja powtarzania jest domyślnie powtarzana.
