Tekst

fillText

na element canvas możemy też wstawiać tekst. Aby to zrobić najpierw musimy zdefiniować czcionkę jakiej chcemy używać. Następnie możemy coś napisać używając metody fillText, która przyjmuje parametry: tekst, x,y i opcjonalnie maksymalna szerokość


const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");   
//ustawienie czcionki  
ctx.font = "60px Arial";
//ustawiene koloru
ctx.fillStyle = "violet"
//wypisanie tekstu
ctx.fillText("Bread",50,50)

StrokeText

analogicznie do fillText występuje metoda strokeText


const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");   
//ustawienie czcionki  
ctx.font = "60px serif";
//ustawiene koloru
ctx.strokeStyle = "blue"
ctx.lineWidth = 3
//obrys tekstu
ctx.fillText("Bread",50,50)

textAlign

podobnie jak w html, w canvasie występuje metoda textAlign. działa ona jednak trochę inaczej, mianowicie określa czy kpolrdynaty podawane przy tworzeniu tekstu odnoszą się do początku teksstu, jego środka czy końca


const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");    
ctx.font = "30px Haveltica";
ctx.fillStyle = "blue"
ctx.textAlign ="center"
ctx.fillText("Center",canvas.width/2, 50)
